# Machine Learning Project

Python 기반 기초 머신러닝 구현 프로젝트입니다.  
머신러닝 모델의 기본 구조를 이해하기 위해 데이터 전처리, 모델 구성, 학습, 예측 과정을 직접 구현하였습니다.

## Project Overview

본 프로젝트는 머신러닝의 기본 개념을 실습하기 위한 저장소입니다.  
Python을 활용하여 데이터 처리와 모델 학습 과정을 구현하고, 머신러닝 모델이 입력 데이터로부터 패턴을 학습하는 흐름을 이해하는 것을 목표로 하였습니다.
